1. Discuss the scope of var, let, and const.


Answer:

In JavaScript, the scope of a variable determines where it can be accessed in the code.
There are three types of variable scope in JavaScript. Those are Global scope, Function scope and Block scope.

Global scope: Variables declared outside of any function have global scope.
They can be accessed from anywhere in the code, including other functions.

Function scope: Variables declared inside a function have function scope.
They can only be accessed from within the function that they are declared in.

Block scope: Variables declared within a block of code, such as a for loop or an if statement,
have block scope. They can only be accessed from within the block that they are declared in.

var is a function scope keyword. let and const those are block scope keyword.

Answer:

To represent the absence of a value. Null and undefined are both used to represent the absence of a value.
However, they have different meanings. Null is used to represent the intentional absence of a value,
while undefined is used to represent the unintentional absence of a value.

3. What do you mean by REST API?


Answer:

REST API stands for Representational State Transfer Application Programming Interface.
It allows applications or devices to communicate with each other using HTTP methods and JSON format.
It follows some principles to make the communication between a client and a server simple and uniform.
For example, a REST API can use HTTP methods like GET, POST, PUT, and DELETE to perform operations on resources.
A resource is anything that can be identified by a URL, such as a user, a product, or a blog post.
